# WeatherAndNote
天气记事本，自动获取每日bing壁纸和天气信息

使用greenDAO  来处理数据库
使用 RxJava +Retrofit 获取天气信息和 bing图片

picasso加载网络图片。

使用RxBus来响应fab点击添加数据后对各个Fragment的影响。

还有DialgFragment 以及CollapsingToolbarLayout的练习
